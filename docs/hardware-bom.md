## 四、CANopen 通信架构

```
                  CAN Bus (500kbps / 1Mbps)
  ════════════════════════════════════════════════
      │            │            │          │
   ┌──┴──┐    ┌───┴───┐   ┌───┴───┐  ┌───┴───┐
   │RPi 4B│    │驱动器1 │   │驱动器2 │  │清洁驱动│
   │+CAN  │    │ID:0x01│   │ID:0x02│  │ID:0x03│
   │ HAT  │    │左轮    │   │右轮    │  │清洁刷 │
   └─────┘    └───────┘   └───────┘  └───────┘
   CANopen
   Master

   - 树莓派作为 CANopen Master
   - 三个驱动器作为 CANopen Slave
   - 通过 SDO 配置参数，PDO 实时控制
```

### 需要从中大力德驱动器手册确认

| 参数 | 用途 |
|------|------|
| CAN 波特率 | 通常 500kbps 或 1Mbps |
| PDO 映射 COB-ID | 速度指令 / 位置反馈 |
| 控制字定义 | 使能、启停、故障复位 |
| 编码器分辨率 | 绝对值编码器位数（如 17-bit） |

## 五、与之前方案的差异总结

| 项目 | 旧方案（小型） | 新方案（工业级） |
|------|--------------|-----------------|
| 重量 | ~2kg | **120kg+** |
| 驱动功率 | 12V 10W ×2 | **24V 500W ×2** |
| 电池 | 12V 9.8Ah 锂电 | **24V 60Ah 铅酸** |
| 轮胎 | 小型橡胶轮 | **300-8 实心** |
| 底盘 | 亚克力板 | **钢管焊接框架** |
| 通信 | GPIO/PWM | **CANopen 总线** |
| 编码器 | 增量霍尔 | **绝对值编码器** |
| 防撞 | 微动开关 | **工业安全触边** |
| 主回路 | 直接接线 | **接触器+急停+保险** |
