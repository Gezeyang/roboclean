# 🎛️ ZBLD.C20-800LRC 驱动器参考

> 中大力德 C20 系列低压无刷驱动器。本文档汇总 CANopen 集成所需的关键参数。

---

## 1. 型号解析

```
ZBLD . C20 - 800 L R C
 │     │     │   │ │ │
 │     │     │   │ │ └── C: CANopen 通信
 │     │     │   │ └─── R: RS485 (Modbus RTU)
 │     │     │   └───── L: 低压 DC 24V-48V
 │     │     └───────── 800: 最大输出 800W
 │     └─────────────── C20 系列
 └───────────────────── ZD BrushLess Driver
```

| **模拟输入** | AI1(旋钮电位器), AI2(外部 0~5V/10V) |

| **冷却方式** | 自然风冷 |
| **防护等级** | IP20 |
| **保护功能** | 欠压、过压、过流、过载、堵转、短路、缺相、过温 |

## 3. 寄存器映射（完整，源自说明书第 6.5 节）

### 3.1 控制寄存器 (2000h~2012h)

| 地址 | 名称 | 说明 | R/W |
|------|------|------|-----|
| `2000h` | 控制命令 | 0001h=正转, 0002h=反转, 0003h=正转点动, 0004h=反转点动, **0005h=停机**, **0006h=紧停**, 0007h=故障复位, 0009h=刹车停机 | W/R |
| `2001h` | 设定转速 | 0~3000 RPM | W/R |
| `2002h` | 电机极对数 | 1~20（默认5） | W/R* |
| `2003h` | 加速时间 | 1~6000（单位：0.1S） | W/R |
| `2004h` | 减速时间 | 1~6000（单位：0.1S） | W/R |
| `2005h` | 控制模式 | 0=测试, 1=开环, 2=闭环, 3=拨码 | W/R* |
| `2006h` | 运行指令选择 | 0=键盘, 1=端子, **2=通讯**, 3=拨码 | W/R |
| `2007h` | 速度给定选择 | 0=键盘, 1=AI1旋钮, 2=AI2外部, **3=Modbus通讯**, 4=多段速, 5=拨码 | W/R |
| `2008h` | 通讯地址 | 1~127（Node ID），0=广播 | W/R* |
| `2009h` | 波特率 | 0=1200~6=57600（默认4=19200） | W/R* |
| `200Ah` | 虚拟输入端子 | 0x000~0x1FF | W/R |
| `200Bh` | 虚拟输出端子 | 0x00~0x0F | W/R |
| `200Eh` | 功能码写使能 | 0=禁止, 1=允许 | W/R |
| `200Fh` | 恢复默认值 | 0=无操作, 1=恢复功能码, 2=恢复CANopen, 3=恢复全部 | W/R* |
| `2012h` | 停机方式 | 0=减速停车, 1=自由停车 | W/R |

### 3.2 状态寄存器 (2100h~2103h)

| 地址 | 名称 | 说明 | R/W |
|------|------|------|-----|
| `2100h` | 状态字1 | 0001h=正转运行, 0002h=反转运行, 0003h=停机, 0004h=故障, 0005h=OFF, 0006h=刹车 | R |
| `2101h` | 状态字2 | Bit0=母线电压, Bit4=过载, Bit5-6=控制源(00=键盘/01=端子/10=通讯), Bit7=键盘连接 | R |
| `2102h` | 故障代码 | 见故障代码表 | R |
| `2103h` | 驱动器识别 | C20系列 = **0x0020** | R |

### 3.3 运行数据 (2200h~)

| 地址 | 名称 | 说明 | R/W |
|------|------|------|-----|
| `2200h` | 设定频率 | 单位 0.1Hz | R |
| `2201h` | 实际频率 | 单位 0.1Hz | R |
| `2202h` | 实际转速 | 单位 RPM | R |
| `2203h` | 实际电流 | 单位 0.1A | R |
| `2204h` | 母线电压 | 单位 0.1V（**可用于电池监测**） | R |
| `2205h` | 模块温度 | 单位 ℃ | R |

## 4. CANopen 集成要点

### 4.1 启用 CANopen

必须通过键盘设置功能码：

| 功能码 | 设置值 | 说明 |
|--------|--------|------|
| `F08.13` | **1** | 选择 CANopen 协议（默认0=CAN Modbus） |
| `F08.00` | 通讯地址 (1~127) | Node ID |
| `F08.01` | 波特率 (0~7) | 0=10K~6=500K, **7=1M** |

### 4.2 PDO COB-ID 对照表（源自说明书第 8 章）

| PDO | COB-ID | 通信参数 | 映射参数 |
|-----|--------|----------|----------|
| RPDO1 | `200h + NodeID` | 1400h | 1600h |
| RPDO2 | `300h + NodeID` | 1401h | 1601h |
| RPDO3 | `400h + NodeID` | 1402h | 1602h |
| RPDO4 | `500h + NodeID` | 1403h | 1603h |
| TPDO1 | `180h + NodeID` | 1800h | 1A00h |
| TPDO2 | `280h + NodeID` | 1801h | 1A01h |
| TPDO3 | `380h + NodeID` | 1802h | 1A02h |
| TPDO4 | `480h + NodeID` | 1803h | 1A03h |
| T_SDO | `580h + NodeID` | — | 1200h |
| R_SDO | `600h + NodeID` | — | 1200h |
| NMT/Heartbeat | `700h + NodeID` | — | 1016h, 1017h |

> 举例：NodeID=2 的 TPDO1 COB-ID = `180h + 2 = 182h`

### 4.3 三个驱动器的规划

| 驱动器 | Node ID | RPDO1 COB-ID | TPDO1 COB-ID | 用途 |
|--------|---------|-------------|-------------|------|
| 左轮 | 1 | 201h | 181h | 速度指令 + 位置反馈 |
| 右轮 | 2 | 202h | 182h | 速度指令 + 位置反馈 |
| 清洁刷 | 3 | 203h | 183h | 速度指令 + 状态反馈 |

### 4.4 通信参数

```
波特率:   500 kbps (F08.01=6) 或 1 Mbps (F08.01=7)
Node ID:  1/2/3（通过 F08.00 设置）
PDO传输:  异步 254/255（推荐），通过事件计时器触发
CAN终端:  总线两端各 120Ω，驱动器内置可配置
```

### 4.5 树莓派连接

```
树莓派 4B
  │
  ├── SPI ── CAN HAT (MCP2515) ── CAN Bus ── 驱动器
  │
  └── 软件: Python canopen 库
```

### 4.6 Python 控制示例

```python
import canopen

# 初始化 CAN 网络
network = canopen.Network()
network.connect(channel='can0', bustype='socketcan')

# 添加三个驱动器
left  = network.add_node(1, 'ZBLD_C20_800LRC.eds')   # 左轮
right = network.add_node(2, 'ZBLD_C20_800LRC.eds')   # 右轮
brush = network.add_node(3, 'ZBLD_C20_800LRC.eds')   # 清洁刷

# 启动所有节点
for node in [left, right, brush]:
    node.nmt.state = 'OPERATIONAL'

# 设置控制模式为通讯控制
left.sdo[0x2006].raw = 2   # 运行指令 = 通讯
left.sdo[0x2007].raw = 3   # 速度给定 = Modbus通讯

# 正转运行 1000 RPM
left.sdo[0x2001].raw = 1000
left.sdo[0x2000].raw = 0x0001  # 正转

# 读取状态
status = left.sdo[0x2100].raw  # 状态字1
speed  = left.sdo[0x2202].raw  # 实际转速
voltage = left.sdo[0x2204].raw # 母线电压 (0.1V)
current = left.sdo[0x2203].raw # 实际电流 (0.1A)

# 紧停
left.sdo[0x2000].raw = 0x0006
```

## 6. 注意事项

1. **48V 系统**：本系统使用 48V，驱动器输出可达 800W，500W 电机在余量内
2. **功能码 F08.13**：必须设为 1 才能启用 CANopen（默认是 CAN Modbus）
3. **控制模式**：运行指令(2006h)和速度给定(2007h)需分别设为"通讯"
4. **CAN 终端电阻**：CAN Bus 两端各需 120Ω
5. **三个驱动器 Node ID**：出厂默认=1，需手动设为 1/2/3（不冲突）
6. **电机霍尔线**：务必正确连接（5-Pin 霍尔端子），否则无法换相
7. **恢复默认**：200Fh=2 可恢复 CANopen 参数（包括 COB-ID、Node ID 等）
8. **故障处理**：读 2102h 获取故障码，2000h=0007h 复位故障
