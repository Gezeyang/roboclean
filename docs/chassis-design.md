## 四、重心与稳定计算

### 4.1 各部件重量及位置

| 部件 | 重量(kg) | X(前后) | Y(左右) | Z(高度) |
|------|---------|---------|---------|---------|
| 电池 48V 60Ah | 55 | 中心偏前 200mm | 中心 | 底板+50mm (最低) |
| 方钢管车架 | 35 | 中心 | 中心 | 底板+150mm |
| 底板 5mm | 25 | 中心 | 中心 | 0 (参考面) |
| 左驱动总成 (电机+减速箱+轮) | 18 | 后轴 | -325mm | 轴心 190mm |
| 右驱动总成 | 18 | 后轴 | +325mm | 轴心 190mm |
| 驱动器×3 | 6 | 中后 | 左右各1 + 中1 | 底板+300mm |
| 推料刷总成 | 12 | 前端 | 中心 | 底板+200~500mm |
| 树莓派+CAN HAT+接线 | 3 | 中 | 中心 | 底板+250mm |
| 万向轮 | 5 | 前端 | 中心 | 接触面 |

### 4.2 重心位置

```
总重 ≈ 177kg (含余量)

前后方向 (X):
  支点: 后轴中心 (驱动轮)
  前部力矩 = 55×200 + 12×850 + 5×850 = 11000 + 10200 + 4250 = 25450 N·mm
  后部力矩 = 18×0 + 18×0 + 6×(-200) = -1200 N·mm
  重心 X ≈ 25450 / 177 ≈ 144mm (后轴前方 144mm)

  稳定裕度: 轴距 850mm, 重心离后轴 144mm
  → 前翻临界坡度: arctan(144/190) ≈ 37% ✓ (远大于 20% 需求)

左右方向 (Y):
  对称布局 → 重心在左右中心

高度 (Z):
  最重件电池离地 80+50=130mm
  重心 Z ≈ 250mm (估算)
  → 侧翻临界角: arctan(325/250) ≈ 52° ✓
```

### 4.3 结论

| 指标 | 计算值 | 要求 | 判定 |
|------|--------|------|:---:|
| 前翻临界坡度 | 37% | ≤20% | ✅ |
| 侧翻临界角 | 52° | >30° | ✅ |
| 驱动轮载荷 | ~120kg (68%) | 单轮 ≥120kg | ✅ |
| 万向轮载荷 | ~57kg (32%) | 单轮 ≥80kg | ✅ |

## 五、驱动轮安装

### 5.1 300-8 轮胎参数

| 参数 | 数值 |
|------|------|
| 外径 | ~380mm (装车后实测) |
| 宽度 | ~100mm |
| 轮毂孔 | 5 螺栓 PCD ~100mm (实物测量) |
| 轴孔 | 需配驱动轴径 |

### 5.2 电机安装座设计

中大力德 500W 减速电机法兰：

```
电机输出轴 (需实物测量法兰尺寸):

       ┌─────────────────┐
       │   ○     ○     ○  │  ← 安装螺栓孔 ×4
       │                   │     孔距: 测实物
       │       ┌───┐       │
       │       │ 轴 │       │  ← 输出轴 (配键槽)
       │       └───┘       │
       │   ○           ○  │
       └─────────────────┘

安装座: 10mm 钢板激光切割
  - 按法兰孔位开 4×M10 通孔
  - 中心开轴孔 (比轴径 +2mm 间隙)
  - 焊接到车架后横梁上
```

### 5.3 驱动轴

```
驱动轴: 45# 钢光轴
  轴径: 按轮毂内孔配 (预估 Ø30-35mm)
  长度: ~200mm (轮毂宽度 + 轴承座厚度 + 余量)
  键槽: 按电机输出轴键槽配 (预估 8×7mm)

轴承座: UCP 系列带座轴承
  淘宝搜索: "UCP轴承座 轴径30"
  数量: 2 个 (左右各一，支撑驱动轴外端)
```
