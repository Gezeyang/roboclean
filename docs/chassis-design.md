# 🔩 车架设计 v3 — 旋转外筒式推料机器人

> 参照 Sveaverken RoboPusher Nimbo 设计
> 核心概念: 外筒旋转推料，内车架固定不动，驱动轮从底部伸出

---

## 一、工作原理

```
不是带个刷子推料，而是整个机身旋转把饲料推出去。

      旋转外筒 (Ø500mm, 长1000mm)
         ← 逆时针旋转 (俯视)
    ╔════════════════════════════════╗
    ║                              ░║  ← 筒体外表面 (带螺旋纹路)
    ║   ┌────────────────────┐    ░║     旋转时把饲料推向围栏方向
    ║   │    内部车架(固定)   │    ░║
    ║   │                    │    ░║
    ║   │  ┌────┐  ┌──────┐ │    ░║
    ║   │  │电池│  │树莓派│ │    ░║
    ║   │  │55kg│  │+驱动 │ │    ░║
    ║   │  └────┘  └──────┘ │    ░║
    ║   │                    │    ░║
    ║   │  ┌─────┐  ┌─────┐ │    ░║
    ║   │  │左电机│  │右电机│ │    ░║
    ║   │  │500W │  │500W │ │    ░║
    ║   │  └──┬──┘  └──┬──┘ │    ░║
    ║   └─────┼────────┼────┘    ░║
    ╚═════════╪════════╪══════════╝
              │        │
         ┌────┴──┐ ┌───┴────┐
         │左驱动轮│ │右驱动轮│  ← 从筒底伸出
         │ 300-8 │ │ 300-8  │
         └───────┘ └────────┘

饲料在地面，筒体旋转 → 筒体表面摩擦饲料 → 把饲料推向围栏
```

### 4.3 外筒驱动

```
外筒旋转驱动:
  200W 电机 (Node ID 3, 中大力德)
  → 减速机 (速比 ~10:1)
  → 小链轮 (Ø80, 20齿)
  → 链条 (08B 滚子链)
  → 大链轮 (Ø320, 80齿, 固定在筒体后端法兰)
  → 筒体旋转 800 RPM

  传动比 = 80÷20 = 4:1 (链轮)
  总减速比 ≈ 10:1 × 4:1 = 40:1

  电机 3000 RPM → 输出轴 300 RPM → 链轮减速 → 筒体 75 RPM...

  不对! 筒体需要 800 RPM。重新算:
  电机额定 3000 RPM → 1:1 直驱链轮 → 筒体 3000÷4 = 750 RPM ≈ 800 ✓

  所以减速箱不需要大速比，大约 1:1 或小速比即可。
  链轮 4:1 减速，电机 3000÷4 = 750 RPM。

固定方式:
  - 200W 电机安装在内部车架后部
  - 链轮传动，链条在筒体后端外部
  - 大链轮螺栓固定在筒体后端法兰上
```

## 八、筒体底部开口与密封设计

这是整台机器最关键的密封点——筒体每分钟旋转 800 圈，驱动轮和内部车架保持不动，开口处必须挡住饲料碎屑。

### 8.1 问题分析

```
三个矛盾:
  1. 外筒 │ → 旋转 (800 RPM)
     内架 │ → 不动
     → 筒体与车架之间有相对运动，不能简单"堵死"

  2. 轮子需要伸出接触地面
     → 必须有开口，不能完全封闭

  3. 牛场饲料 = 碎草+玉米粉+粉尘
     → 极其容易钻进缝隙，进了筒内就积在电池和驱动器上
```

### 8.2 密封方案：三道防线

```
              旋转方向 →→→
    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 外筒底部
            ┌──────────────┐
    第一道: │  挡料板(固定)  │  ← 焊在内部车架上，不随筒体转
            │  斜面向外导流  │     饲料碰到挡板 → 向外滑落
            └──────────────┘
            ┌──────────────┐
    第二道: │  毛刷密封条    │  ← 装在筒体开口内缘，随筒旋转
            │  尼龙刷毛 3排  │     刷毛扫过挡板表面，刮掉残留
            └──────────────┘
            ┌──────────────┐
    第三道: │  橡胶唇口      │  ← 装在内部车架上，向外翻
            │  柔性接触      │     筒体转动时唇口始终贴合筒内壁
            └──────────────┘
    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
              │           │
           ┌──┴──┐     ┌──┴──┐
           │左轮 │     │右轮 │  ← 从三道防线下方伸出
           └─────┘     └─────┘
```

```

### 8.6 第三道防线 — 橡胶唇口

```
橡胶唇口 (装在内部车架上，向外翻压住筒体内壁):

    筒体内壁
    ━━━━━━━━━━━━━━

三层的分工:
  挡料板 → 挡住大颗粒饲料 (碎草、玉米粒)
  毛刷   → 扫掉粘在挡料板上的细碎料
  唇口   → 最后的防线，挡住粉尘进入筒内

### 8.8 维护检查

```
日常检查 (每天):
  - 目视检查开口处是否有饲料堆积
  - 清理挡料板上可能卡住的长草

每周检查:
  - 检查毛刷刷毛磨损情况 (刷毛短于 15mm 需更换)
  - 检查橡胶唇口是否有裂口
  - 检查螺栓是否松动

每月检查:
  - 拆卸毛刷座清理内部
  - 检查筒体内壁是否有饲料进入痕迹
  - 如有 → 检查唇口贴合是否紧密，调整压条
```

## 九、装配顺序

```
第1步: 筒体制作
  - 2mm 钢板卷圆 → 焊接纵缝 → Ø500×1000mm 圆筒
  - 两端焊接法兰 (8mm 环形板)
  - 筒体外表面焊接 6 条螺旋纹路 (6mm 圆钢)
  - 筒体底部开驱动轮窗口 + 折边

第2步: 主轴 + 车架
  - 主轴 Ø40 穿入前后轴承座
  - 焊接方钢管车架在主轴上
  - 安装左右驱动电机座板
  - 安装电池托架 + 设备安装板

第3步: 车架装入筒体
  - 车架从筒体端面推入
  - 前端轴承座螺栓固定到筒体前端法兰
  - 后端轴承座(含大链轮)固定到筒体后端法兰
  - 驱动轮从底部开口伸出

第4步: 外筒驱动
  - 安装 200W 电机到车架后端
  - 装小链轮 → 套链条 → 张紧
  - 大链轮螺栓固定到筒体后法兰

第5步: 电气
  - 装电池 (主轴下方)
  - 装驱动器 ×3 (主轴上方, 散热好)
  - 装树莓派 + CAN HAT
  - 接线、布管

第6步: 行走系统
  - 装左右驱动轮 → 联轴器 → 电机
  - 装前万向轮 (主轴前端下方)

第7步: 防锈 + 测试
  - 所有焊接处打磨
  - 喷防锈底漆 + 面漆
  - 筒体旋转测试 (空载 800 RPM)
  - 行走测试
```
