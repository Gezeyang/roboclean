## 七、装配顺序 (给焊接店)

```
第1步: 车架焊接
  - 纵梁 F1+F2 平放地面，标出横梁位置
  - 焊接前后中三根横梁 F3/F4/F5 → 形成矩形框架
  - 检查对角线长度差 <3mm (保证方正)
  - 焊接电池托架 F6+F7 到底板
  - 焊接电机座板 F9+F10 到后横梁
  - 焊接万向轮座板 F11 到前横梁
  - 焊接 8 块三角筋

第2步: 底板固定
  - 底板 F8 放上车架
  - 用 8×M12 螺栓固定 (不用焊接，便于日后拆卸维修)

第3步: 行走系统安装
  - 安装万向轮到 F11 (4×M12)
  - 安装轴承座到后横梁下方
  - 穿入驱动轴 → 套联轴器 → 装轮子
  - 安装电机，对齐联轴器

第4步: 电气安装
  - 电池放入托架，压板固定
  - 安装 3 台驱动器 (侧挂)
  - 安装树莓派 + CAN HAT (电池上方空间)
  - 安装 LiDAR (前上方)
  - 接线 (按 [hardware-bom.md](hardware-bom.md) §2)

第5步: 推料刷
  - 从 F3 前横梁上方安装转轴
  - 安装斜撑 → 装滚筒
  - 调整角度 60°，弹簧拉紧

第6步: 表面处理
  - 所有焊缝打磨
  - 喷防锈底漆 (铁红)
  - 喷面漆 (灰色/蓝色)
```
