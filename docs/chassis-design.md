# 🔩 车架设计 v3 — 旋转外筒式推料机器人

> 参照 Sveaverken RoboPusher Nimbo 设计
> 核心概念: 外筒旋转推料，内车架固定不动，驱动轮从底部伸出

---

## 一、工作原理

```
不是带个刷子推料，而是整个机身旋转把饲料推出去。

      旋转外筒 (Ø500mm, 长1000mm)
         ← 逆时针旋转 (俯视)
    ╔════════════════════════════════╗
    ║                              ░║  ← 筒体外表面 (带螺旋纹路)
    ║   ┌────────────────────┐    ░║     旋转时把饲料推向围栏方向
    ║   │    内部车架(固定)   │    ░║
    ║   │                    │    ░║
    ║   │  ┌────┐  ┌──────┐ │    ░║
    ║   │  │电池│  │树莓派│ │    ░║
    ║   │  │55kg│  │+驱动 │ │    ░║
    ║   │  └────┘  └──────┘ │    ░║
    ║   │                    │    ░║
    ║   │  ┌─────┐  ┌─────┐ │    ░║
    ║   │  │左电机│  │右电机│ │    ░║
    ║   │  │500W │  │500W │ │    ░║
    ║   │  └──┬──┘  └──┬──┘ │    ░║
    ║   └─────┼────────┼────┘    ░║
    ╚═════════╪════════╪══════════╝
              │        │
         ┌────┴──┐ ┌───┴────┐
         │左驱动轮│ │右驱动轮│  ← 从筒底伸出
         │ 300-8 │ │ 300-8  │
         └───────┘ └────────┘

饲料在地面，筒体旋转 → 筒体表面摩擦饲料 → 把饲料推向围栏
```

### 4.3 外筒驱动

```
外筒旋转驱动:
  200W 电机 (Node ID 3, 中大力德)
  → 减速机 (速比 ~10:1)
  → 小链轮 (Ø80, 20齿)
  → 链条 (08B 滚子链)
  → 大链轮 (Ø320, 80齿, 固定在筒体后端法兰)
  → 筒体旋转 800 RPM

  传动比 = 80÷20 = 4:1 (链轮)
  总减速比 ≈ 10:1 × 4:1 = 40:1

  电机 3000 RPM → 输出轴 300 RPM → 链轮减速 → 筒体 75 RPM...

  不对! 筒体需要 800 RPM。重新算:
  电机额定 3000 RPM → 1:1 直驱链轮 → 筒体 3000÷4 = 750 RPM ≈ 800 ✓

  所以减速箱不需要大速比，大约 1:1 或小速比即可。
  链轮 4:1 减速，电机 3000÷4 = 750 RPM。

固定方式:
  - 200W 电机安装在内部车架后部
  - 链轮传动，链条在筒体后端外部
  - 大链轮螺栓固定在筒体后端法兰上
```

### 6.2 结论

| 指标 | 计算值 | 要求 | 判定 |
|------|--------|------|:---:|
| 重心高度 | ~筒体中心偏下 | 低于筒体中心 | ✅ 电池拉低重心 |
| 前翻临界坡度 | >30% | ≤20% | ✅ 驱动轮在后，重心偏后 |
| 侧翻临界角 | >45° | >30° | ✅ 轮距650mm, 重心低 |
| 驱动轮载荷 | ~65% 总重 | 单轮≥120kg | ✅ |
| 万向轮载荷 | ~35% 总重 | 单轮≥80kg | ✅ |

## 九、装配顺序

```
第1步: 筒体制作
  - 2mm 钢板卷圆 → 焊接纵缝 → Ø500×1000mm 圆筒
  - 两端焊接法兰 (8mm 环形板)
  - 筒体外表面焊接 6 条螺旋纹路 (6mm 圆钢)
  - 筒体底部开驱动轮窗口 + 折边

第2步: 主轴 + 车架
  - 主轴 Ø40 穿入前后轴承座
  - 焊接方钢管车架在主轴上
  - 安装左右驱动电机座板
  - 安装电池托架 + 设备安装板

第3步: 车架装入筒体
  - 车架从筒体端面推入
  - 前端轴承座螺栓固定到筒体前端法兰
  - 后端轴承座(含大链轮)固定到筒体后端法兰
  - 驱动轮从底部开口伸出

第4步: 外筒驱动
  - 安装 200W 电机到车架后端
  - 装小链轮 → 套链条 → 张紧
  - 大链轮螺栓固定到筒体后法兰

第5步: 电气
  - 装电池 (主轴下方)
  - 装驱动器 ×3 (主轴上方, 散热好)
  - 装树莓派 + CAN HAT
  - 接线、布管

第6步: 行走系统
  - 装左右驱动轮 → 联轴器 → 电机
  - 装前万向轮 (主轴前端下方)

第7步: 防锈 + 测试
  - 所有焊接处打磨
  - 喷防锈底漆 + 面漆
  - 筒体旋转测试 (空载 800 RPM)
  - 行走测试
```
